# Glory Gym

**Glory Gym** is a sleek, AI-inspired workout suggester built to engineer your results. Whether you are a beginner looking to build muscle or an advanced athlete aiming to lose fat, Glory Gym provides personalized workout routines tailored to your fitness level, goals, and available equipment.

## 🚀 Features

- **Personalized Workout Suggestions**: Select your fitness level, primary goal, available equipment, and duration to receive a custom workout routine.
- **Glassmorphism UI**: A highly modern, sleek user interface featuring glass-like cards, glowing accents, and animated floating particles.
- **Light/Dark Mode Toggle**: Seamlessly switch between a stealthy dark mode and a crisp light mode. Your preference is automatically saved in your browser's local storage.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices, ensuring a great experience anywhere.
- **Frozen Compact Footer**: A fixed, unobtrusive footer that stays anchored at the bottom of the screen while you scroll.
- **Interactive Controls**: Range sliders for workout duration and dynamic frequency selection buttons.

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure.
- **CSS3 / Variables**: Custom properties, flexbox, grid, glassmorphism effects, keyframe animations, and modern responsive layouts.
- **Vanilla JavaScript**: DOM manipulation, dynamic content rendering, and `localStorage` for theme preferences (No external JS frameworks required).
- **Bootstrap 5**: Used strictly for rapid grid layout and responsive structural classes.
- **Bootstrap Icons**: Crisp, scalable vector icons.

## 💻 Getting Started

To run this application locally, you do not need any build tools or local servers.

1. Clone or download this repository to your local machine.
2. Navigate to the `glory-gym` folder.
3. Open the `index.html` file in your preferred modern web browser (e.g., Chrome, Firefox, Edge, Safari).
